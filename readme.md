# tuya_zigbee_gw_ext_sdk

## overview

feature :

* Register to Tuya Cloud platform.
* Realtime preview via Tuya Smart APP

## how to use

### run a quick demo

1) edit Makefile, and set COMPILE_PREX ?= to absolute path of your toolchain gcc.

2) compile one application.

```bash
make
```
